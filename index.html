Getting started with Skill Engine

In order to use Skill Engine, you must include the JavaScript and CSS file on your website. You can get these files built for you from many different locations.

Downloading the code locally

In some situations, you can't use Skill Engine from a CDN and you must include the files through your own static file servers.

  1. Download the code from GitHub and copy the dist directory to your project.
  2. Include the following lines of code in the <head> section of your HTML. 
    <link href="path/to/skillEngine.Grid.min.css" rel="stylesheet" />
<script src="path/to/skillEngine.Grid.min.js"></script>
  3. Check out the example pages to start using the additional features of Skill Engine.
